# JobFinder Portal

A full-stack job search application built using Flask, Snowflake, HTML, CSS, and REST APIs.

## Features

* Search jobs using a public Jobs API
* Store search history in Snowflake
* Save favorite jobs
* Analytics dashboard
* Search history tracking
* CSV export functionality
* Dark mode support
* Location-based filtering
* Duplicate prevention

## Tech Stack

* Python
* Flask
* Snowflake
* HTML
* CSS
* JavaScript
* REST API

## How to Run

1. Create virtual environment
2. Install requirements
3. Configure Snowflake credentials
4. Run:

python app.py

5. Open:

http://127.0.0.1:5000
